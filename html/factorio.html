What is Factorio?

Factorio is a game about building factories. You have to gather resources and automatize production, so you can
launch a rocket and win the game. The game released officialy on August 14th, 2020, after being in early access
for four years. Below is the trailer for the 1.0 release.

Main features

Below is a list of the main features of Factorio:

  • Expansive randomly generated map
  • Deep crafting and logistic systems
  • Tower-defense styled combat
  • Online multiplayer
  • Extensive mod support
